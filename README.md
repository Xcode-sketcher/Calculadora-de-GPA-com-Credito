# Calculadora de GPA com Crédito

Projeto desenvolvido em C# para calcular o GPA (Grade Point Average) com base nas notas das disciplinas e creditos extras.
Em seeguida, o GPA calculado é exibido no console.

Este projeto foi criado como parte de um exercício de aprendizado em C#.

Disponibilizado sob a licença MIT.
## Funcionalidades
- Cálculo do GPA com base nas notas fornecidas.
- Exibição do GPA calculado no console.
- Validação básica de entrada para garantir que as notas estejam dentro do intervalo válido (0-100).
- Suporte para crédito extras que adicionam 10% em cima da nota.
